Gestão de Processos

A PD Case está organizada para atender as diferentes demandas do mercado para desenvolvimento de software.

Processo Padrão de Desenvolvimento de Software

A PD Case possui hoje um processo bem estruturado, claro, ágil, seguro e que também é continuamente melhorado visando a geração eficiente de produtos considerando os critérios de qualidade, prazos de entrega e gestão eficiente do controle de mudanças.

Estrutura do Processo Padrão para Desenvolvimento de Softwares

O processo de desenvolvimento da PD Case atende todas as áreas de qualidade exigidos pelo MPS-br nível C, sendo baseado no guia de gerenciamento de projetos PMBOK-PMI, o que atesta a utilização das boas práticas em gerenciamento de projetos amplamente utilizadas no mercado.
